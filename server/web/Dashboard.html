機敏資料去識別化

本工具目的為將機敏資料集進行去識別化程序，以達到隱私保護之效益；其中功能包括：
1. 使用者可選擇隱私層級以達到欲想之隱私程度。
2. 使用者可選擇欲計算之欄位。
3. 使用者可選擇欲有關聯性之欄位。
4. 使用者可自定義欄位區間數。
5. 使用者可藉由條件查詢功能比較與原始資料之誤差率。
6. 使用者可藉由資料分析功能比較與原始資料之統計差異性。
